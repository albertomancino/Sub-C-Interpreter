PROGRAM NODE

typedef struct ProgramNode {
  FunNodeList * list;
  struct TreeNode * scope;
  struct ScopeStack * ActualScope;
} ProgramNode;

Un ProgramNode (Tree.h) è una struttura che contiene
1 FunNodeList denominata list
2 TreeNode * denominato scope
3 ScopeStack * denominato ActualScope

1 La lista di nodi funzione è una struttura (Tree.h) che ha il comportamento logico
di una lista composta da elementi di tipo FunNode

2 Lo scope è un TreeNode (parser.h) che conterrà un riferimento alla simbol table globale.

3 ActualScope è uno ScopeStack (parser.h), quindi una pila di Scope (parser.h),
ed è la pila degli scope in modo da gestire l'accesso alle variabili correttamente
